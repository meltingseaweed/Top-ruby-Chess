# Top-ruby-Chess
Create a chess game that can be played in the terminal/console. All pieces should follow the basic rules of chess. Special rules may be added after, such as castling. Project's purpose is to tackle a very broad problem with lots of components to challenge the skills learned up until now.

Review: Overall the project was difficult but rewarding. This is my first project with such a broad scale and little startup knowledge and help, so it helped me gain experience doing a big project with many pieces from scratch. I think I got my first understanding of the importance of scalability, and depending on how the code is setup or written can really affect how hard or easy it is to make changes or add new aspects or rules later on. For example, adding castling, en passant, etc.
